# Task: consolidate_day_report

## Objetivo
Gerar fechamento analitico do dia consolidando todos os trades processados.

## Entradas
- `analise_consolidada.md` de cada trade
- `metadados_trade.json` de cada trade

## Saidas
- `fechamento_dia.md`
- `resumo_quantitativo_dia.json`
- `resumo_qualitativo_dia.json`
- `pendencias_dia.json`
- `indice_trade_reports.json`

## Criterio de conclusao
- Fechamento diario completo com pendencias explicitas.
